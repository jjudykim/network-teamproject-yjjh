### 🔹 **에디터 창 구성 이해 (Scene, Game, Hierarchy 등)**

---

유니티는 다양한 창으로 구성되어 있으며, 각 창은 역할이 뚜렷함

영화 촬영 현장에 비유해 이해해보자!

- **Scene = 영화 세트장**
    - 배우(오브젝트), 조명, 소품 등을 배치하는 공간
    - 예: 큐브 오브젝트를 배치하면 이곳에 나타남
- **Hierarchy = 세트장 내 오브젝트 목록**
    - Scene에 배치된 모든 오브젝트가 트리 형태로 나열됨 (계층 구조)
    - 보이지 않는 오브젝트(예: GameObject만 있는 빈 오브젝트)도 포함됨
- **Game = 영화 관객이 보게 될 실제 화면**
    - Main Camera가 촬영하는 화면을 그대로 보여줌
    - 관객은 오직 카메라를 통해 본 화면만 보게 됨
- **Project = 영화의 소품 창고**
    - 모델, 음악, 텍스처 등 모든 리소스가 저장됨 (단, 직접 Scene에 배치해야 실제 등장).
    - Assets 폴더가 루트.
- **Inspector = 오브젝트 세부 정보**
    - 선택한 오브젝트의 구성요소(Component)를 보여주고 설정 가능
    - 예: Transform(Position, Rotation, Scale), Mesh Renderer, Collider 등
- **Console = 로그, 에러 메시지 출력**

### 🔹 **Scene 조작 방법 (like FPS 게임 방식)**

---

- **기본 이동**
    - `마우스 우클릭 + WASD`: FPS 스타일 이동
    - `Q / E`: 위/아래로 이동
- **시점 조작**
    - `마우스 우클릭`: 시선(고개 돌리기) 조작
    - `Alt + 마우스 왼쪽`: 피벗 기준으로 회전
    - `Alt + 마우스 오른쪽`: 줌인/아웃
- **마우스 휠 조작**
    - 그냥 휠 돌리기: 줌
    - `우클릭 + 휠`: 카메라 이동 속도 조절

<br>
<br>

### 🔹 **오브젝트 조작 단축키 (QWER 툴)**

---

- `Q`: View Tool (기본 시점 조작 모드)
- `W`: Move Tool (이동)
    - X, Y, Z 축 기준 이동 가능
    - 정사각형 핸들 클릭 시, 2축 고정 이동 가능
- `E`: Rotate Tool (회전)
    - 원 형태의 핸들로 회전 가능
- `R`: Scale Tool (크기 조절)
    - 각 축 기준으로 늘리거나 줄이기

📌 조작 시 Inspector의 Transform(Position, Rotation, Scale) 값이 함께 변함

### 🔹 **카메라 뷰 조정**

---

- 메인 카메라는 실제 촬영에 사용하는 카메라
- **Ctrl + Shift + F**: 현재 Scene 뷰 카메라 위치를 Main Camera에 반영
    - 촬영 감독이 보고 있던 화면을 실제 카메라 각도로 설정 가능

<br>
<br>

### 🔹 **기타 팁**

---

- **Ctrl + Z**: 작업 되돌리기
- Scene 뷰에서 자유롭게 놀아보며 조작법 익히는 게 가장 빠름
- 궁극적으로 **우리는 감독**, 게임은 **영화**, Main Camera는 **촬영 카메라**라는 사고방식이 중요
