### 🔹 Terrain 실습 요약

---

**Terrain이란?**

**Terrain**은 Unity에서 넓은 지형(땅)을 표현하기 위해 제공하는 **전용 3D 오브젝트 시스템**

기존의 `Plane` 오브젝트보다 **유연하고 확장성이 뛰어난 지형 도구**

→ 오픈월드 게임이나 자연 환경이 필요한 씬에서 자주 사용

<br>

**Terrain의 주요 특징**

| 기능 | 설명 |
| --- | --- |
| ✅ **지형 조절 기능** | **높이 조절(Raise/Lower)**, 구릉/산/계곡 등을 자유롭게 조형 |
| ✅ **텍스처 페인팅(Paint Texture)** | 땅 위에 **다양한 질감 텍스처**를 칠할 수 있음 |
| ✅ **나무/잔디 심기(Paint Trees/Details)** | 자연 환경 요소(나무, 풀 등)를 브러시처럼 쉽게 배치 |
| ✅ **Collider 포함** | 생성 시 **Terrain Collider** 자동 부착 → 캐릭터가 충돌 가능 |
| ✅ **Terrain Layer 사용** | 텍스처, 풀 등은 **Terrain Layer**로 관리 → 외부 에셋 연동도 용이 |

<br>

**Terrain 생성 방법**

```
Hierarchy → Right Click → 3D Object → Terrain
```

<br>

**기본 구조**

- `Terrain` 컴포넌트
- `Terrain Collider` 자동 포함
- 생성 시 `New Terrain Data` 리소스도 함께 생성됨

**Terrain Layer 페인팅**

1. `Terrain 컴포넌트 → Paint Terrain → Paint Texture`
2. `Edit Terrain Layers → Create Layer`
3. 텍스처 선택 후 칠하기 가능

**브러시 설정**
- **Brush Size / Opacity / Brush Shape**
- ➕ 그림판과 유사한 인터페이스

**다운로드 에셋 분석**

- `_TerrainAutoUpgrade`: Terrain Layer 리소스 파일
- `Demo_Terrain`: Terrain 데이터 프리셋

### 🔹 광원의 종류

---

| 광원 종류 | 설명 |
| --- | --- |
| **Directional Light** | 태양처럼 방향만 중요, 위치 영향 없음 |

### 🔹 Lightmap 개념 및 Baking

---

**조명 모드**

- `Realtime`: 실시간 연산 (CPU/GPU 부하 큼)
- `Baked`: 미리 계산한 값으로 그림자 표현
- `Mixed`: 정적 오브젝트는 Bake, 나머지는 실시간

**Static 체크**

- 해당 오브젝트가 **움직이지 않음**을 명시
- 체크 시 자동으로 라이트맵 계산 시작됨

**자동 Baking 설정 해제**

```
Window → Rendering → Lighting Settings → Auto Generate (Off)
```

**수동 베이킹**

- `Generate Lighting` 버튼 클릭
- 완료 후 Scenes 하위에 라이트맵 폴더 생성됨

### 🔹 Demo 씬으로 이사 작업

---

**기존 Game 씬 → Demo로 마이그레이션**

1. 필요한 객체 (예: 유니티짱, Main Camera 등) 복사
2. `Demo.unity` 씬에 붙여넣기
3. 기존 Game 씬 삭제
4. `Demo` 씬을 `Game` 이름으로 저장 (Ctrl + Shift + S)

> 최종 개발 씬을 마무리할 기반 환경이 준비된 것!
>
