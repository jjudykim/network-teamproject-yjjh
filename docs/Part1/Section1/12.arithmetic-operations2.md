### 복합 연산자

---

`a = a + 5` 와 같이, 본인이 포함된 연산의 결과를 다시 본인에 할당하는 경우 연산자를 축약할 수 있다

→ `a += 5`

모든 산술 연잔자 적용 가능!

ex) `a *= b;`, `a /= 3;`

<br>
<br>

### 증감 연산

---

1을 증가시키거나 감소시키는 작업을 간단한 연산자로 표현 가능 → `++`, `--`

- `+= 1`의 축약형이라고 볼 수 있음

전위 증감 연산자 (피연산자의 앞에 오는지)인지, 후위 증감 연산자 (피연산자의 뒤에 오는지)인지에 따라 결과가 달라질 수 있다

- 전위 증감 연산자 : `++a` ⇒ a에 1을 더한 후 해당 라인의 명령을 실행
- 후위 증감 연산자 : `a++` ⇒ 해당 라인의 모든 명령을 실행 후 a에 1을 더함
