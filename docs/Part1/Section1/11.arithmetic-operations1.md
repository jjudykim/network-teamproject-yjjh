이제, 데이터의 가공에 대해 알아보자

몬스터와의 전투를 통해 데미지를 입어 HP가 닳거나,

아이템을 사용해 다시 HP가 오르는 등의 데이터 가공이 이루어져야 한다

→ 산술 연산이 필요한 단계!

<br>
<br>

### 산술 연산

---

산술 연산자들 → `+` `-` `*` `/` `%`

```csharp
int hp = 100;
int damage = 5;
int heal = 3;
hp = hp - damage;
hp = hp + heal; 
```

산술 연산자들 사이에는 우선순위라는 것이 존재하기 때문에, 우선순위에 의해 연산 순서가 결정됨

→ 인위적으로 괄호`()`를 사용함으로써 연산 순서를 임의로 설정할 수 있음

일반 사칙연산에서 보지 못한 연산자 `%` : 나머지 연산자

두 수를 나누기를 했을 때, 발생하는 나머지를 구하는 연산자

```csharp
if (n % 2 == 0) // 2로 나누었을 때 0이 남는지? 1이 남는지?
```

→ 이런 식으로 짝/홀 검사를 하는 등의 로직에서 활용
