### 데이터를 어떻게 저장할까?

---

RAM이 들고 있는 모든 정보들은 전기신호 유무로만 모든 것을 표현할 수 있기 때문에, 010101과 같은 형태로만 표현

→ 즉, 전기 신호가 있으면 1, 아니면 0으로 하나의 상태를 표현

그럼 동일하게 0101001010 이란 상태가 있다고 하더라도, 그 값을 우리가 어떤 관점으로 바라볼 것인지에 따라 의미가 완전히 달라질 수 있음

즉, 데이터를 다룰 때는 어떤 형식의 데이터를 다룰 것인지를 먼저 정한 다음 작업을 진행해야 함!

<br>
<br>

### 데이터 형식의 대표적인 4가지

---

- int : 정수형 데이터(-1, 0, 1, 6)
- float : 실수형 데이터(3.14)
- string : 문자열 데이터(”Rookiss”)
- bool : 불리언 데이터 (true/false)

→ 각 적재적소 맞는 데이터를 찾아, 어느 데이터를 사용해 표현해야 하는지 고려하기
