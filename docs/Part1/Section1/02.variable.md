### 변수

---

우리는 결국 RAM이라는 부품에, 우리가 원하는 데이터를 올려 어느 공간을 임대해 사용함으로써 데이터를 관리하거나 사용할 수 있게 된다

이때 고려해야 할 것 → RAM에도 다 주소값이 존재!

이때 편리하게 사용하기 위해 사용하는 것이 **변수**

변수를 정의함으로써

- 어떤 용도의 데이터를 저장할 것인지 (타입)
- 데이터 바구니를 어떤 이름으로 부를 것인지 (이름)
- 데이터 바구니에 어떤 데이터를 넣을 것인지 (값)

→ 이런 바구니는 RAM의 한 주소를 차지해, 주소값을 모르더라도 바구니 이름으로 데이터를 사용할 수 있게 됨

바구니(변수)를 사용하는 3가지 단계

1. 데이터를 할당하는 단계 → 변수 선언

   : 변수를 선언함으로써 데이터를 할당할 공간을 마련

2. 데이터를 저장하는 단계 → 데이터 할당

   : 변수에 원하는 데이터 값을 할당

3. 데이터를 꺼내오는 단계 → 변수 읽기

   : 변수로 저장해둔 데이터에 접근 가능 (읽기/쓰기 가능)

<br>

**변수의 선언**

> [데이터 타입] [이름];
>

```csharp
int hp;
```

<br>

**변수에 데이터를 할당할 때**

> [이름] = ?;
>

```csharp
hp = 100;
int maxHP = 500;    // 선언과 동시에 초기화
```

<br>


**변수의 데이터를 읽을 때**

> [이름];
>

```csharp
Console.WriteLine(hp);
```
