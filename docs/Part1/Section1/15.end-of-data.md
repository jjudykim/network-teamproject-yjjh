### 정리

---

우리가 프로그래밍을 한다

→ CPU와 RAM에 이런저런 일감을 던져주는 작업인 것인데…

CPU : 연산 담당

RAM : 저장소 담당

따라서 데이터, 로직이 핵심!

변수 : 데이터를 다룰 수 있는 바구니의 역할

변수 사용의 3가지 단계

1. 임대
2. 할당
3. 사용

이 데이터를 산술연산으로 인해 가공할 수 있음!

또 가공된 데이터를 비교 연산 & 논리 연산을 통해 특정 상황에 대한 로직을 설계할 수 있음

반드시 기억해야 할 데이터 타입 4가지

→ `int`, `float`, `string`, `bool`

추가로 배울 `var`은 자동추론을 통해 모든 타입을 담을 수 있게됨

다만, 데이터 타입을 명시하는 것이 변수를 사용하는 용도가 정확히 드러나고 가독성이 좋음!
