### 스트링 포맷

---

int ↔ string과 같이

일반적인 정수형, 실수형 타입과 문자열 타입을 서로 변환하는 과정은 간단하지 않음

→ string은 태생적으로 클래스에 해당하기 때문에, 조금 더 복잡하다

만약 서로를 변환하고 싶다면?

**string → int**

```csharp
string input = Console.ReadLine();
// 방법1
int i1 = int.Parse(input);
// 방법2
int i2  = Convert.ToInt32(input);
```

**int → string**

```csharp
// 방법1 (string Format)
string message1 = string.Format("당신의 HP는 {0} 입니다", intHP);
Console.WriteLine(message1);

// 방법2 (string Interpolation)
string message2 = $"당신의 HP는 {intHP}입니다."
Console.WriteLine(message2);
```
