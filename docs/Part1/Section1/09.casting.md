### casting (캐스팅)

---

casting, 캐스팅은 형식 변환을 의미한다 (타입 변환)

- 캐스팅 문법을 이용해 인위적으로 형식 변환을 할 수 있다
- 큰 곳에서 작은 곳으로든, 작은 곳으로든 큰 곳으로든 할 수 있다

  (ex. long → int, short → int)


⇒ 단, 경우에 따라서 일부 데이터가 소실될 수 있기 때문에 주의해야 한다

```csharp
short a = 100;
int b = a;          // (자동 형변환)

float l = 3.14f;
int i = f;          // 보통 오류 발생! (데이터 소실의 위험)
int i = (int)f;     // 캐스팅 문법 활용 
```
