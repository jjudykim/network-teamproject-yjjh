### 논리연산자

---

논리 연산이 왜 필요할까?

→ 여러가지 조건을 동시에 만족하는지를 판별하거나, 조건 여러가지를 혼합해 판단해야 하는 경우가 발생한다

조건을 어떻게 조합을 해서, 상황을 판단해 로직을 실행시키는지를 결정하는 경우의 수가 굉장히 많아진다

이를 위해 논리 연산자를 사용!

- AND `&&` : 두 가지를 동시에 만족해야 True, 하나라도 아니라면 False

    ```csharp
    true && true => true
    true && false => false
    false && false => false
    ```

- OR `||` : 두 가지 중 하나라도 만족하면 True, 아니면 False

    ```csharp
    true || true => true
    true || false => true
    false || false => false
    ```

- NOT `!` : 피연산자를 반전시킴!

    ```csharp
    !true => false
    !false => true
    !(true && true) => false
    !(false || false) => true
    ```


논리연산자 사용의 예

```csharp
int level = 1;
int gold = 100;

// 레벨 5 이상이고, gold 100 이상만 입장 가능
bool canEnter = (level >= 5) && (gold >= 100);
Console.WriteLine(canEnter);        // False
```
