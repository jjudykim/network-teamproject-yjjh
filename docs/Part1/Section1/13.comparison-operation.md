### 비교 연산자

---

- 작다 `<`
- 작거나 같다 `<=`
- 같다 `==`
- 크거나 같다 `>=`
- 크다 `>`
- 다르다 `!=`

비교연산자의 결과값은 bool타입이다 (참 / 거짓의 형태)

```csharp
int hp = 100;

bool isFullHp = (hp == 100);
bool isAlive = (hp > 0);
```

```csharp
int level = 10;

bool canEnterDungeon = (level <= 5);
Console.WriteLine(canEnterDungeon);   // False
```
