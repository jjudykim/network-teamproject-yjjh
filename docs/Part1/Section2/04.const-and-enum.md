### 상수

---

방금 제작한 가위바위보에서, 가위-바위-보가 0, 1, 2를 뜻하는게 번거롭고 헷갈린다!

→ 따라서 이를 변수로 명명하는 방법도 있고, 상수를 사용하는 방법도 있는데 …

Switch-case문에서 case로 사용되는 값은 변수로 값이 변경될 수 없고, 고정된 값이어야 한다!

따라서 이때 상수를 사용해 case를 정의

```csharp
const int ROCK = 1;
```

의 형태로 상수 선언 가능

### 열거형

---

이때 int 문법을 사용해 묶게되면 각각이 별개인 느낌이 드는데,

사실 가위-바위-보는 한 그룹으로 생각하는 것이 더 편하다!

이렇게 공통점이 있는 묶음을 지칭하는 것을 → `enum` (열거형)으로 해보자

```csharp
enum Choice
{
	Rock = 1,
	Paper = 2,
	Scissors = 0,
}
```

→ 이렇게 상수값 설정 가능
