### while 문

---

while문의 사용 방법

```csharp
while( )    // 괄호 안의 내용이 참이라면 문을 실행
{
            // 중간에 누군가가 멈춰주지 않을 때까지 (혹은 괄호 안의 내용이 거짓이 될 때까지)
            // 무한 반복
}
```

- 무한루프가 돌지 않도록 탈출 조건을 잘 설정하는 것이 중요

ex) While문으로 Hello World 5번 출력

```csharp
int count = 5;

while(count > 0)
{
	Console.WriteLine("Hello World");
	count--;
}
```

### do while문

---

```csharp
do
{
	// 이 안에 있는 내용을 무조건 1번은 일단 실행!
	// 그리고 괄호 안의 내용이 참이라면 계속해서 실행
} while();
```

ex) do-While문으로 강사님이 잘생겼는지 확인하기 (답정너)

```csharp
string answer;

do
{
	Console.WriteLine("강사님은 잘생기셨나요? (y/n) : ");
	answer = Console.ReadLine();
} while(answer != "y");

Console.WriteLine("강사님은 잘생겼어요!");
```

→ 강사님이 잘생겼다고 말하지 않으면… 영원히 물어봄…
