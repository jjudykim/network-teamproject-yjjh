### 조건을 case로 나눌 수 있는 swtich문

---

> switch(검사할 변수) <br>
{ <br>
case 케이스1: // 케이스1일 경우 진행 <br>
break; <br>
case 케이스2: // 케이스2일 경우 진행 <br>
break; <br>
… <br>
default: 
// 아무 케이스에도 해당이 되지 않을 때 진행 <br>
}
>
- if-else → switch-case으로 언제는 형태를 변경할 수 있음
- switch-case → if-else 는 항상 성립하지는 않음!
- 조금 더 가독성이 좋다는 장점이 있지만, 활용할 수 있는 경우가 보다 한정적임

if-else문으로 작성했던 가위바위보 코드를 switch-case 문으로 변경해보자

```csharp
namespace CSharp
{
	class Program
	{
		static void Main(string[] args)
		{
			int choice = 0;        // 0 : 가위, 1 : 바위 : 2 : 보
			
			switch(choice)
			{
				case 0:
					Console.WriteLine("가위입니다.");
					break;
				case 1:
				  Console.WriteLine("바위입니다.");
				  break;
				case 2:
					Console.WriteLine("보입니다.");
					break;
				default:
					Console.WriteLine("가위바위보가 아닌데요?!");
					break;
			}
		}
	}
}
```

### 삼항 연산자

---

if-else문을 간단하게 한 줄로 표현 가능한 연산자로, 조건의 결과에 따라 return되는 값이 달라짐

> 조건 `?` true일 경우 return `:` false일 경우 return
>

```csharp
namespace CSharp
{
	class Program
	{
		static void Main(string[] args)
		{
			int number = 25;
			
			bool isPair;
			
			// if-else 문으로 표현
			if ((number % 2) == 0)
				isPair = true;
			else
				isPair = false;
				
			// 같은 식을 삼항 연산자를 활용해 표현
			isPair = (number % 2) == 0 ? true : false;
		}
	}
}
```
