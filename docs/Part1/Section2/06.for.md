### 🔹 **For문 개요**

---

- `while`문과 같은 반복문 중 하나.
- **초기화식**, **조건식**, **반복식**을 한 줄에 작성하여 반복 로직을 더 명확하게 표현할 수 있음.
- 주로 정해진 횟수만큼 반복할 때 사용.

<br>
<br>

### 🔹 **기본 문법**

---

```csharp
for (초기화식; 조건식; 반복식)
{
    // 반복할 코드
}
```

- **초기화식**: 반복문이 시작될 때 한 번만 실행됨.
- **조건식**: 반복 여부를 결정하는 조건. `false`가 되면 반복 종료.
- **반복식**: 반복할 때마다 실행되는 코드 (일반적으로 변수 증가/감소).

<br>
<br>

### 🔹 **For문과 While문의 비교**

---

| 구분 | While문 | For문 |
| --- | --- | --- |
| 선언 위치 | 변수와 증가식을 별도로 선언해야 함 | 한 줄에 모든 반복 요소를 작성 |
| 코드 가독성 | 길어질 경우 증가식 실수 가능 | 반복 구조가 한눈에 보임 |
| 사용 용도 | 반복 횟수가 정해지지 않은 경우에 적합 | 정해진 횟수 반복 시 유용 |

**💡 동일한 기능을 하는 예제**

```csharp
// While문
int count = 0;
while (count < 5)
{
    Console.WriteLine("Hello World");
    count++;
}

// For문
for (int i = 0; i < 5; ++i)
{
    Console.WriteLine("Hello World");
}
```

- `while`문에서는 변수 선언과 증가를 개별적으로 처리해야 하므로 실수할 가능성이 있음.
- `for`문은 반복 로직이 한 줄로 정리되어 가독성이 뛰어남.

<br>
<br>

### 🔹 **For문 실행 순서**

---

```csharp
for (int i = 0; i < 5; ++i)
{
    Console.WriteLine("Hello World");
}
```

1️⃣ **초기화식 실행**: `int i = 0;`

2️⃣ **조건식 검사**: `i < 5` 참이면 실행, 거짓이면 종료

3️⃣ **코드 블록 실행**: `Console.WriteLine("Hello World");`

4️⃣ **반복식 실행**: `i++` 후 다시 2️⃣로 이동

⏩ **반복 흐름:**

```
i = 0 → 조건 검사 → 출력 → i 증가 (i=1) → 조건 검사 → 출력 → i 증가 ... 반복
```

<br>
<br>


### 🔹 **for문에서 초기화/조건/반복식 생략 가능**

---

```csharp
int i = 0;
for (; i < 5; )  // 조건만 사용 가능
{
    Console.WriteLine("Hello World");
    i++; // 수동 증가 필요
}
```

- 모든 요소를 생략하면 무한 루프가 됨.

```csharp
for (;;)
{
    Console.WriteLine("무한 루프");
}
```

<br>
<br>


### 🔹 **0부터 시작하는 이유**

---

- 일반적으로 프로그래머들은 반복문을 **0부터 시작**하는 것이 익숙함.
- 예를 들어 5번 반복하려면 보통 `i < 5`로 작성.

```csharp
for (int i = 0; i < 5; i++) // 0,1,2,3,4 → 총 5번 실행
```

- `i = 1; i <= 5;`로도 구현 가능하지만 일반적이지 않음.

<br>
<br>


### 🔹 **주의할 점**

---

1️⃣ 조건을 잘못 설정하면 무한 루프 발생

```csharp
for (int i = 0; i >= 0; i++) // i가 계속 증가하여 절대 false가 되지 않음
{
    Console.WriteLine("무한 루프");
}
```

2️⃣ 조건이 거짓이면 한 번도 실행되지 않음

```csharp
for (int i = 0; i < 0; i++)
{
    Console.WriteLine("출력되지 않음");
}
```

- `i = 0`, `0 < 0`이 `false`이므로 한 번도 실행되지 않음.

<br>
<br>


### 📌 **정리**

---

- `for`문은 `while`문보다 가독성이 뛰어나고 반복 로직을 명확하게 표현할 수 있음.
- **초기화식 → 조건식 → 반복식** 순서로 실행됨.
- 정해진 횟수만큼 반복할 때 가장 많이 사용됨.
- 무한 루프에 주의해야 하며, 실행 순서를 정확히 이해하는 것이 중요.
