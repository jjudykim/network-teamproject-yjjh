### 🔹 **메소드란?**

---

- **코드 재사용**을 위해 특정 기능을 묶어서 정의하는 것.
- `Console.WriteLine()` 같은 기능도 내부적으로 메소드로 정의됨.
- 메소드 = C++의 함수(Function)과 같은 개념.

<br>
<br>

### 🔹 **메소드 기본 문법**

---

```csharp
한정자 반환형식 메소드이름(매개변수 목록)
{
    // 실행할 코드
}
```

**예제) HelloWorld 출력 메소드**

```csharp
static void HelloWorld()
{
    Console.WriteLine("Hello World");
}
```

✅ 설명

- `static` → 클래스에서 직접 호출 가능 (객체 생성 불필요).
- `void` → 반환값 없음.
- `HelloWorld()` → 메소드 이름.
- `()` → 매개변수 없음.
- `{}` → 실행될 코드 블록.

**메소드 호출 방법**

```csharp
HelloWorld();         // 같은 클래스 내에서 호출
Program.HelloWorld(); // 클래스명.메소드명() 형태로 호출 가능
```

<br>
<br>

### 🔹 **입력(매개변수)과 출력(반환값)**

---

✅ **반환값이 있는 메소드**

```csharp
static int Add(int a, int b)
{
    return a + b;
}
```

✅ **호출 방법**

```csharp
int result = Add(4, 5);
Console.WriteLine(result);  // 출력: 9
```

**설명**

- `int Add(int a, int b)` → 두 개의 `int`를 받아 `int` 반환.
- `return`을 사용해 결과를 호출한 곳으로 반환.

<br>
<br>


### 🔹 **값 복사(Pass by Value) vs 참조 전달(Pass by Reference)**

---

✅ **값 복사 (Pass by Value)**

```csharp
csharp
복사편집
static void AddOne(int number)
{
    number = number + 1;
}

static void Main()
{
    int a = 0;
    AddOne(a);  // 복사된 값 전달
    Console.WriteLine(a);  // 출력: 0 (원본 변경 X)
}

```

→ 설명

- 기본적으로 매개변수는 **값(짭퉁, 복사본)**이 전달됨.
- 즉, `number`는 `a`의 복사본이므로 `a`는 변경되지 않음.\

✅ **참조 전달 (Pass by Reference)**

```csharp
static void AddOne(ref int number)
{
    number = number + 1;
}

static void Main()
{
    int a = 0;
    AddOne(ref a);  // 참조 전달
    Console.WriteLine(a);  // 출력: 1 (원본 변경 O)
}
```

→ 설명

- `ref` 키워드를 사용하면 **실제 변수의 메모리 주소(진퉁)**를 전달.
- 즉, `number`는 `a`를 직접 참조하므로 `a`의 값이 변경됨.

### 📌 **정리**

---

- **메소드**: 특정 기능을 묶어 코드 재사용성을 높임.
- **반환값 O (`return`)**: 계산 결과를 호출한 곳으로 반환 가능.
- **값 복사 (기본 방식)**: 매개변수를 복사하여 전달 → 원본 변경 X.
- **참조 전달 (`ref`)**: 원본 변수 자체를 전달 → 원본 변경 O.
- **C++의 함수와 동일한 개념**이며, `static` 키워드는 클래스 단위에서 호출할 때 사용됨.
